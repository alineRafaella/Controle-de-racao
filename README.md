# *Sistema de Gestão de Estoque de Ração*

Este projeto foi desenvolvido para controle de insumos de rações em canis, garantindo que nunca falte alimento para os animais. Através de análise do consumo diário e da definição de ponto de ressuprimento, o sistema monitora os níveis de estoque e gera alertas visuais (gráfico) para identificar o momento de novas compras.

### Principais funcionalidades:
- Cálculo de consumo diário e mensal.<br/>
- Visualização de dados com linha de limite (ponto de ressuprimento).

### Ferramentas utilizadas:
- Python: para análises de dados e visualizações.<br/>
- Pandas: manipulação e exploração de dados.<br/>
- Numpy:  operar funções matemáticas.<br/>
- Matplotlib: criação de gráficos para visualização.<br/>
